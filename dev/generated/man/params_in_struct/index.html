Parameters

NamedTrajectory.jl support passing parameters as a Tuple when construct a NamedTrajectory.

using NamedTrajectories

First we need to define number of timesteps and timestep

T = 10
dt = 0.1
0.1

then build named tuple of components and data matrices.

components = (
    x = rand(3, T),
    u = rand(2, T),
    Δt = fill(dt, 1, T),
)
(x = [0.4062552993399964 0.47409436234174707 … 0.46787732841580765 0.7532628740294747; 0.9120928036925167 0.319983373170177 … 0.8990950163698239 0.3658111103012295; 0.2858198146251171 0.3751549762356047 … 0.1992400594262136 0.6808455777896475], u = [0.8746347514594355 0.8372920421891435 … 0.0612024911300556 0.6870462440238744; 0.5999661068184766 0.036326252969435924 … 0.15005176080775806 0.5507821460057825], Δt = [0.1 0.1 … 0.1 0.1])

we must specify a timestep and control variable for the trajectory.

timestep = 0.1
control = :u
:u

some global params as a NamedTuple

params = (
    α = rand(1),
    β = rand(1)
)
(α = [0.46402521658991536], β = [0.33590762488063153])

we can now create a NamedTrajectory object with parameters specification.

traj = NamedTrajectory(components; timestep=timestep, controls=control, global_data=params)
