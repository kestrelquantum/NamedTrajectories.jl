Add params in NamedTrajectory

NamedTrajectory.jl support passing parameters as a Tuple when construct a NamedTrajectory.

using NamedTrajectories

First we need to define number of timesteps and timestep

T = 10
dt = 0.1
0.1

then build named tuple of components and data matrices.

components = (
    x = rand(3, T),
    u = rand(2, T),
    Δt = fill(dt, 1, T),
)
(x = [0.974129051157997 0.47385349458743165 … 0.39751459684349644 0.5392110698860862; 0.6460940599213243 0.03958860724661484 … 0.5985232174603142 0.6625160711284003; 0.9782466381800168 0.6158479482000226 … 0.9167448685779608 0.4912736324335102], u = [0.2837107106645351 0.5152254399229611 … 0.7681824844179395 0.23892023511791816; 0.5233875680412838 0.06293093388300142 … 0.6994586324168551 0.12583159766715624], Δt = [0.1 0.1 … 0.1 0.1])

we must specify a timestep and control variable for the trajectory.

timestep = 0.1
control = :u
:u

some global params as a NamedTuple

params = (
    α = rand(1),
    β = rand(1)
)
(α = [0.5909347466746028], β = [0.25646274248440715])

we can now create a NamedTrajectory object with parameters specification.

traj = NamedTrajectory(components; timestep=timestep, controls=control, params=params)
